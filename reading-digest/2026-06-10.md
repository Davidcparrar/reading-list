## 1. One Token per Multimodal Evidence: Latent Memory

- **Authors:** Zhi Zheng, Ziqiao Meng, Hao Luan, et al.
- **Summary:** Proposes a retrieval-augmented memory paradigm where each piece of
  retrieved evidence — text snippet or image — is compressed into a single
  high-dimensional latent token rather than re-injected into the prompt as raw
  context. The generator consumes these compact tokens directly, matching
  competitive RAG-baseline QA accuracy on text and multimodal benchmarks while
  spending 3×–10× fewer generator tokens at inference.
- **Theme:** Memory for LLMs and agents — latent-space long-term / retrieval
  memory.
- **Relevance:** Direct hit on theme (1) and a clean efficiency story: pushes the
  memory representation from "more raw context" toward learned latent tokens,
  which is the front line of agent memory research right now.
- **Link:** https://arxiv.org/abs/2606.10572

## 2. Attention Amnesia in Hybrid LLMs: When CoT Fine-Tuning Breaks Long-Range Recall

- **Authors:** Xinyu Zhou, Boyu Zhu, Yi Xu, et al.
- **Summary:** Diagnoses a systematic failure mode in hybrid linear-attention LLMs:
  chain-of-thought fine-tuning quietly degrades long-context recall even as
  reasoning scores improve. Introduces QK-Restore, a training-free patch that
  reinstates the original query/key projections in the attention layers, lifting
  S3@256K from 65.4% to 76.4% while preserving the CoT reasoning gains.
- **Theme:** Memory for LLMs and agents — long-range working memory in hybrid
  architectures.
- **Relevance:** Useful negative-result-plus-fix for anyone deploying hybrid /
  linear-attention LLMs as agent backbones; the failure mode is silent in
  standard reasoning evals.
- **Link:** https://arxiv.org/abs/2606.11052

## 3. Data Journalist Agent: Transforming Data into Verifiable Multimodal Stories

- **Authors:** Kevin Qinghong Lin, Batu EI, Yuhong Shi, et al.
- **Summary:** Builds a multi-agent pipeline (Data2Story) that assigns specialized
  roles for extracting figures from raw data, drafting prose, and laying out
  charts/images, with each numeric claim linked back to a source row or external
  reference. Produces evidence-grounded multimodal articles rather than the
  plain-text defaults of single-shot LLMs, and reports competitive,
  evidence-traceable output on data-journalism tasks.
- **Theme:** Multi-agent frameworks & orchestration — role-specialised agents
  with provenance constraints.
- **Relevance:** Concrete real-world multi-agent system whose orchestration choices
  (role separation, citation-binding) are reusable patterns for agentic LLM
  apps outside journalism.
- **Link:** https://arxiv.org/abs/2606.11176

## 4. SearchSwarm: Towards Delegation Intelligence in Agentic LLMs

- **Authors:** Pu Ning, Quan Chen, Kun Tao, et al.
- **Summary:** Targets long-horizon search/browse tasks by training the model to
  produce "harness-guided trajectories" that decompose work and delegate
  subtasks to subagents, keeping individual context windows tractable. The
  resulting SearchSwarm-30B-A3B reaches 68.1 on BrowseComp and 73.3 on
  BrowseComp-ZH, the best among open models at comparable scale.
- **Theme:** Multi-agent frameworks & orchestration — explicit delegation /
  subagent coordination as a learned skill.
- **Relevance:** One of the cleaner empirical demonstrations that delegation
  itself can be optimised (not just hand-coded), with strong long-horizon
  browsing numbers.
- **Link:** https://arxiv.org/abs/2606.09730

## 5. Role-Agent: Bootstrapping LLM Agents via Dual-Role Evolution

- **Authors:** Xucong Wang, Ziyu Ma, Shidong Yang, et al.
- **Summary:** Uses a single LLM in two roles simultaneously — the agent solving
  tasks and the environment generating them — so the model can co-evolve by
  practicing on environments it itself produces. The World-In-Agent /
  Agent-In-World loop concentrates training on the agent's current weak
  patterns and yields an average gain of >4 points over strong baselines on
  agent benchmarks.
- **Theme:** Multi-agent frameworks & orchestration — self-play / dual-role
  co-evolution.
- **Relevance:** A cheap self-play recipe that does not require a separate
  environment model; directly relevant to agent post-training pipelines.
- **Link:** https://arxiv.org/abs/2606.10917

## 6. Workflow-GYM: Long-Horizon Evaluation of Computer-Use Agentic Tasks

- **Authors:** Liya Zhu, Jingzhe Ding, Jian Zhang, et al.
- **Summary:** Introduces a benchmark of long-horizon GUI workflows in
  professional-software domains. Even the strongest current agents land only
  slightly above 30% task success, and the per-error breakdown isolates four
  recurring failure modes — workflow-stage omission, error propagation,
  objective drift, and shallow understanding of the target software — that
  generic agent metrics miss.
- **Theme:** Multi-agent frameworks & orchestration — long-horizon agent
  evaluation harness.
- **Relevance:** Provides a sharper, mode-specific yardstick for computer-use
  agents; the failure taxonomy is directly actionable for orchestration design.
- **Link:** https://arxiv.org/abs/2606.11042

## 7. EEVEE: Towards Test-Time Prompt Learning

- **Authors:** Weixian Xu, Shilong Liu, Mengdi Wang, et al.
- **Summary:** Test-time prompt-learning framework for LLM agents that must handle
  heterogeneous input streams. A learned router partitions incoming queries
  into task clusters and co-evolves with a pool of cluster-specific prompts,
  so the right prompt is chosen and refined per input at deployment time.
  Reports +10.38 and +24.32 average points across multi-benchmark suites over
  no-routing baselines.
- **Theme:** Multi-agent frameworks & orchestration — router-plus-prompts
  orchestration at test time.
- **Relevance:** Connects directly to mixture-of-prompts / mixture-of-agents
  research; the router-prompt co-evolution recipe is a clean baseline for
  agent-stack designers.
- **Link:** https://arxiv.org/abs/2606.11182

## 8. Kwai Keye-VL-2.0 Technical Report

- **Authors:** Kwai Keye Team, Bin Wen, Changyi Liu, et al.
- **Summary:** Open-source Mixture-of-Experts multimodal foundation model aimed at
  long-video understanding and agentic use. Adapts DeepSeek-style sparse
  attention onto a GQA backbone to give lossless 256K-token context, and uses
  Cross-Modal Multi-Teacher On-Policy Distillation plus Context-RL and
  Video-RL to combat catastrophic forgetting across the multi-task alignment
  stages.
- **Theme:** Core LLM advances — long-context multimodal backbone for agents.
- **Relevance:** Substrate-level paper: an openly-released 256K-context
  multimodal MoE is exactly the kind of backbone the agent / memory work
  above will build on.
- **Link:** https://arxiv.org/abs/2606.10651

## 9. DRPO: Rethinking the Divergence Regularization in LLM RL

- **Authors:** Jiarui Yao, Xiangxin Zhou, Penghui Qi, et al.
- **Summary:** Replaces the hard token-masking trust-region used in off-policy
  LLM RL with a smooth, advantage-weighted quadratic regulariser on the policy
  shift. The continuous penalty preserves trust-region geometry while giving
  bounded, attenuated gradients on diverging updates, so off-policy data is
  damped instead of clipped away.
- **Theme:** Core LLM advances — RL post-training relevant to agentic / reasoning
  systems.
- **Relevance:** A small, plug-in change to a fast-moving area (RLVR / off-policy
  LLM RL); likely to slot straight into existing reasoning-agent training
  pipelines.
- **Link:** https://arxiv.org/abs/2606.09821
