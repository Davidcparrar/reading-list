## 1. OPD-Evolver: Cultivating Holistic Agent Evolver via On-Policy Distillation
- Authors: Guibin Zhang, Xun Xu, Yanwei Yue, et al.
- Summary: The paper argues that storing trajectories is not the same as learning to evolve, and proposes a slow–fast co-evolution loop in which a fast agent reads/uses/writes/maintains a four-level memory hierarchy at test time while a slow loop distills those four memory operations into the deployable policy via outcome-calibrated attribution and privileged hindsight. The reported gains are up to +11.5% over memory baselines such as ReasoningBank and roughly +5.8% over Skill0-style training on multi-domain agent benchmarks.
- Theme: Memory for LLMs and agents.
- Relevance: Treats memory as a learnable, end-to-end competence (selection, action, writing, maintenance) rather than a passive store — directly central to theme 1.
- Link: https://arxiv.org/abs/2606.17628

## 2. ActWorld: From Explorable to Interactive World Model via Action-Aware Memory
- Authors: Zhexiao Xiong, Yizhi Song, Hao Kang, et al.
- Summary: Existing interactive world models compress history with a recency bias and therefore "forget" the event-transition frames that determine future object states; ActWorld replaces that with a hierarchical action-aware memory that routes compression by interaction importance and adds a persistent bank of event-update and object-identity tokens across long rollouts. A new 100K interaction-video dataset with per-chunk chain-of-thought captions is used to train the chunk-autoregressive model and shows higher interaction fidelity than navigation-only baselines.
- Theme: Memory for LLMs and agents (memory architecture for long-horizon generative agents).
- Relevance: Concrete, novel mechanism for action-aware long-horizon memory — directly maps to long-term/episodic memory and state management.
- Link: https://arxiv.org/abs/2606.17730

## 3. VisualClaw: A Real-Time, Personalized Agent for the Physical World
- Authors: Haoqin Tu, Jianwen Chen, Zijun Wang, et al.
- Summary: VisualClaw is a self-evolving VLM agent built on two ideas — a cascaded hybrid encoder that drops low-information frames and hot/cold compresses a text skill bank for ~98% API-cost reduction, and a skill-evolution loop where retrieved memories of past failures condition an offline evolver that updates the skill bank without touching weights. On a new VisualClawArena benchmark and across two backbones (Gemini 3 Flash, GPT-5.2) it reports +3.85% average gains on video-QA tasks.
- Theme: Memory for LLMs and agents (retrieval-augmented skill memory, failure-driven update).
- Relevance: Weight-free, retrieval-conditioned memory updates are exactly the "retrieval-augmented memory / context-state management" thread of theme 1.
- Link: https://arxiv.org/abs/2606.16295

## 4. LectūraAgents: A Multi-Agent Framework for Adaptive Personalized AI-Assisted Learning and Embodied Teaching
- Authors: Jaward Sesay, Yue Yu, Siwei Dong, et al.
- Summary: A hierarchical multi-agent system with a ProfessorAgent coordinating research, planning, review, and content-delivery sub-agents to produce personalized lectures and execute pedagogically motivated on-slide actions (highlighting, underlining, handwriting). The TASA algorithm uses salience heuristics and temporal semantic segmentation to align teaching actions with speech for each learner profile, evaluated across high-school, undergraduate and graduate levels.
- Theme: Multi-agent frameworks & orchestration.
- Relevance: Clean example of role/tool assignment and coordination protocols across specialized agents — fits theme 2 squarely.
- Link: https://arxiv.org/abs/2606.16428

## 5. ProCUA-SFT Technical Report
- Authors: Jaehun Jung, Ximing Lu, Brandon Cui, et al.
- Summary: To train computer-use agents at scale the authors build a fully automated pipeline where a single VLM (Kimi-K2.5) plays goal generator, precondition judge, and trajectory executor on live desktops seeded with real spreadsheets and presentations, gated by binary feasibility checks. Fine-tuning UI-TARS 7B for one epoch on the resulting 3.1M step-level samples (from 93K trajectories, 2,484 app combos) takes OSWorld from 26.3% to 45.0% — over 35 points above AgentNet-trained baselines.
- Theme: Multi-agent / agentic-system training and evaluation.
- Relevance: Concrete recipe for producing usable agent trajectories with a single-VLM generator/verifier/actor pipeline, removing planner–actor capability gaps.
- Link: https://arxiv.org/abs/2606.17321

## 6. GameCraft-Bench: Can Agents Build Playable Games End-to-End in a Real Game Engine?
- Authors: Tongxu Luo, Rongsheng Wang, Jiaxi Bi, et al.
- Summary: The authors formalize end-to-end game generation as producing a complete playable artifact in a real engine and judge it by Engine Grounding, Artifact Completeness, and Interactive Verification — replaying gameplay and grading with multimodal rubrics. The instantiated benchmark has 140 Godot tasks across 15 game families; the best frontier coding agent reaches only 41.46%, exposing gaps in content depth, visual feedback, and coherent presentation.
- Theme: Multi-agent / agentic-system evaluation.
- Relevance: Cross-cutting agentic-system benchmark that stresses tool use, planning, and engine-grounded verification — useful evaluation methodology for theme 2 work.
- Link: https://arxiv.org/abs/2606.17861

## 7. Zone of Proximal Policy Optimization: Teacher in Prompts, Not Gradients
- Authors: Byung-Kwan Lee, Ximing Lu, Shizhe Diao, et al.
- Summary: ZPPO keeps the teacher inside the prompt instead of the policy gradient: on hard questions where every student rollout fails, it builds a Binary Candidate-included Question pairing one teacher-correct and one student-wrong response and a Negative Candidate-included Question that aggregates the student's wrong rollouts, then recirculates these via a replay buffer until the student's accuracy crosses 0.5. Across Qwen3.5 students from 0.8B–9B with a 27B teacher on 31 VLM/LLM/video benchmarks it outperforms off/on-policy distillation and GRPO, with the largest gains at the smallest scale.
- Theme: Core LLM advances (fine-tuning/PEFT and reasoning).
- Relevance: Practical, transferable post-training recipe for small-student reasoning that directly supports agentic systems built on compact backbones.
- Link: https://arxiv.org/abs/2606.18216

## 8. Rethinking the Role of Efficient Attention in Hybrid Architectures
- Authors: Ziqing Qiao, Yinuo Xu, Chaojun Xiao, et al.
- Summary: A systematic study of hybrids that mix full attention with sliding-window or recurrent mixers finds that long-range retrieval is carried almost entirely by full-attention layers, while efficient attention mostly shapes the optimization trajectory; the authors name a "Large-Window Laziness" effect where bigger SWA windows actually delay retrieval-head formation. Acting on this they apply NoPE only to the full-attention layers of a small-window SWA hybrid and report large long-context gains with negligible short-context cost.
- Theme: Core LLM advances (long-context architecture).
- Relevance: Mechanistic insight on which components carry long-range state — directly relevant to the context/state-management thread of theme 1 as well.
- Link: https://arxiv.org/abs/2606.15378

## 9. VibeThinker-3B: Exploring the Frontier of Verifiable Reasoning in Small Language Models
- Authors: VibeThinker team (authors not listed on HF page).
- Summary: A 3B dense model post-trained with a Spectrum-to-Signal pipeline of curriculum SFT, multi-domain RL, and offline self-distillation reaches 94.3 on AIME26 (97.1 with claim-level test-time scaling), 80.2 Pass@1 on LiveCodeBench v6, and 96.1% on unseen LeetCode contests while keeping 93.4 on IFEval. The authors generalize their earlier 1.5B work into a Parametric Compression-Coverage Hypothesis: verifiable reasoning compresses into a small reasoning core, while open-domain knowledge needs broad parameter coverage.
- Theme: Core LLM advances (reasoning, small-model post-training).
- Relevance: Strong evidence that agentic reasoning backbones can be small — useful as the model layer under memory/multi-agent systems.
- Link: https://arxiv.org/abs/2606.16140

## 10. A Gradient Perspective on RLVR Stability and Winner Advantage Policy Optimization
- Authors: Prasanth YSS, Zhichen Ren, Rasa Hosseinzadeh, et al.
- Summary: The authors analyze RLVR training collapse through a token-level gradient taxonomy that predicts how updates move next-token probabilities and entropy, and show that stability depends jointly on the advantage sign and the token's probability under the current policy. From this they propose Winner Advantage Policy Optimization (WAPO), a clipped update that fires only on positive-advantage completions, and report stable training plus matched-or-better accuracy on math and multi-hop QA benchmarks.
- Theme: Core LLM advances (RL for reasoning / agentic training).
- Relevance: Stable, simple RLVR recipe is a building block for the agent post-training stacks behind themes 1 and 2.
- Link: https://arxiv.org/abs/2606.16154
