# Reading digest — 2026-07-10

## 1. LaMem-VLA: Dual Latent Memory in Vision-Language-Action Models for Robotic Manipulation
- **Authors:** Hongyu Qu, Jianzhe Gao, Xiaobin Hu, et al.
- **Summary:** The paper proposes a latent-memory-native architecture for VLA policies that treats past experience as tokens living inside the same embedding space the model already uses to reason. A curator sorts history into short- and long-term vaults, a seeker retrieves relevant slices given the current observation, and a condenser compresses those slices into memory tokens that a weaver splices in beside the instruction. On SimplerEnv and LIBERO the resulting policy handles temporally dependent, long-horizon tasks that stump Markovian baselines.
- **Theme:** Memory for LLMs and agents
- **Relevance:** Rare example of memory being wired directly into an agent's reasoning stream (not a bolt-on retrieval bank), with an explicit short-/long-term split — directly on-theme for episodic/working-memory work in agents.
- **Link:** https://arxiv.org/abs/2607.07608

## 2. Sparse Delta Memory: Scaling the State of Linear RNNs through Sparsity
- **Authors:** Loïc Cabannes, Pierre-Emmanuel Mazaré, Gergely Szilvasy, et al.
- **Summary:** SDM replaces the dense key–value outer-product update in Gated DeltaNet with sparse addressing into a much larger explicit memory, so hidden-state capacity can grow by orders of magnitude at matched FLOPs. Under isoFLOP training the larger memory measurably improves in-context learning and long-context retrieval, and treating the initialized memory as parametric also lifts common-knowledge and reasoning scores. The result is a concrete route to give linear-attention models transformer-style recall without paying the quadratic bill.
- **Theme:** Memory for LLMs and agents
- **Relevance:** Attacks the long-standing recall gap of linear RNNs by rethinking the memory substrate itself — high-relevance for the memory/state-management thread.
- **Link:** https://arxiv.org/abs/2607.07386

## 3. UniClawBench: A Universal Benchmark for Proactive Agents on Real-World Tasks
- **Authors:** Zhekai Chen, Chengqi Duan, Kaiyue Sun, et al.
- **Summary:** UniClawBench evaluates proactive agents through 400 bilingual real-world tasks that run inside live Docker containers, scored by fine-grained per-step checkpoints instead of a single final-answer bit. Its closed-loop harness pairs an executor agent with a hidden supervisor and a user agent so multi-turn feedback is realistic without leaking grading criteria. By factoring tasks along five capabilities (skill use, exploration, long-context reasoning, multimodal understanding, cross-platform coordination) and swapping base models across multiple agent frameworks, it disentangles model-capability effects from framework-design effects.
- **Theme:** Multi-agent frameworks & orchestration
- **Relevance:** Explicit multi-agent evaluation harness (executor + supervisor + user) plus a clean model-vs-framework ablation — directly targets orchestration and agentic-system evaluation.
- **Link:** https://arxiv.org/abs/2607.08768

## 4. Infinite Worlds with Versatile Interactions (LingBot-World 2.0)
- **Authors:** Zelin Gao, Qiuyu Wang, Jiapeng Zhu, et al.
- **Summary:** LingBot-World 2.0 pushes an interactive world model to unbounded-horizon 720p/60fps rollouts via causal pretraining and a real-time distilled variant. What makes it interesting beyond video is the agentic harness stacked on top: a pilot agent plans and executes character behavior while a director agent synthesizes new environmental elements as the scene unfolds. It is one of the first serious integrations of a role-differentiated multi-agent orchestrator into a generative world simulator.
- **Theme:** Multi-agent frameworks & orchestration
- **Relevance:** Role-and-tool-differentiated pilot/director orchestration inside a live simulator — a fresh datapoint on how multi-agent coordination scales into generative environments.
- **Link:** https://arxiv.org/abs/2607.07534

## 5. Single-Rollout Asynchronous Optimization for Agentic Reinforcement Learning
- **Authors:** Zhenyu Hou, Yujiang Li, Jie Tang, Yuxiao Dong
- **Summary:** SAO drops GRPO-style group sampling in favor of a single rollout per prompt so that asynchronous agentic RL stops paying a stability tax on long-horizon trajectories. A value model and a strict double-sided token-level clip keep updates on-policy enough to train stably for a thousand steps, and the resulting recipe outperforms GRPO/DAPO/GSPO on SWE-Bench Verified, BeyondAIME, and IMOAnswerBench. It is the RL loop actually used to train the open GLM-5.2 (750B-A40B) agentic model.
- **Theme:** Core LLM advances (agentic RL / fine-tuning)
- **Relevance:** Practical, deployed answer to the "GRPO does not fit long-horizon agents" problem — one of the more grounded pieces of agent-RL infrastructure to appear this week.
- **Link:** https://arxiv.org/abs/2607.07508

## 6. Accurate, Interdisciplinary and Transparent Structure-property Understanding with Deep Native Structural Reasoning (SciReasoner)
- **Authors:** Chen Tang, Yizhou Wang, Jianyu Wu, et al.
- **Summary:** SciReasoner is a multimodal scientific foundation model that discretizes coordinates, topologies, and periodic connectivities into a shared structure-aware vocabulary, so structural evidence becomes addressable tokens inside the reasoning chain. Across 86 benchmarks it hits state-of-the-art on 67, notably lifting GO Cellular Component Fmax from 0.42 to 0.55 and single-step retrosynthesis accuracy from 0.63 to 0.72 with fragment-level traces. Expert double-blind evaluation ranks its reasoning traces at least on par with a frontier LLM in 98% of cases.
- **Theme:** Core LLM advances (reasoning)
- **Relevance:** A concrete demonstration that structured, domain-native tokens can turn a scientific reasoner into something inspectable and empirically stronger than a general-purpose LLM — meaningful signal for reasoning-model design.
- **Link:** https://arxiv.org/abs/2607.07708

## 7. Ideas Have Genomes: Benchmarking Scientific Lineage Reasoning and Lineage-Grounded Idea Generation
- **Authors:** Yifan Zhou, Qihao Yang, Yan Li, et al.
- **Summary:** IG-Bench formalizes scientific ideas as typed "Idea Genome" objects and represents relationships between papers as GenomeDiffs covering inheritance, mutation, loss, external import, and novel insertion. It ships 1,961 lineage traces, 1,085 genome objects, and 920 pairwise diffs across 10 domains, split into a 42-task closed-form exam and a lineage-conditioned generation arena scored by a Population-Evolution metric. Across 14 LLM-based "scientists" the top model only reaches 27.3% exact accuracy on lineage reasoning, exposing a real compositional bottleneck in current agentic-research systems.
- **Theme:** Core LLM advances (reasoning / agentic-system evaluation)
- **Relevance:** Rare benchmark that stresses *compositional* reasoning over prior-work structure rather than isolated puzzles — hits the "how do we evaluate research agents" gap directly.
- **Link:** https://arxiv.org/abs/2607.08758

## 8. CausalDS: Benchmarking Causal Reasoning in Data-Science Agents
- **Authors:** Andrej Leban, Yuekai Sun
- **Summary:** CausalDS closes the gap between symbolic causal-reasoning benchmarks (which lack realistic analysis) and data-analysis benchmarks (which lack principled causal structure) by systematically generating novel synthetic causal graphs and pairing them with agentic data-science tasks. LLM agents are evaluated end-to-end: they must combine abstract causal inference with actual tool use over the generated datasets. The result is a diagnostic that separates pattern-matching data-wrangling from genuine causal understanding in current agents.
- **Theme:** Core LLM advances (agentic-system evaluation)
- **Relevance:** Fresh, controllable evaluation surface for causal reasoning inside agentic data-science pipelines — cleaner signal than existing curated causal datasets.
- **Link:** https://arxiv.org/abs/2607.08093
