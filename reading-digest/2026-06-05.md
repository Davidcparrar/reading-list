# Reading digest — 2026-06-05

Found 10 paper(s) in the last 48h window.

## 1. Rethinking Continual Experience Internalization for Self-Evolving LLM Agents
- **Authors:** Jingwen Chen, Wenkai Yang, Shengda Fan, et al.
- **Summary:** The authors stress-test how LLM agents internalize accumulated experiences across iterations and show that naive multi-iteration self-evolution induces capability collapse. They isolate two design axes — principle-level vs. instance-level memory and step-wise vs. global injection — and report that principle-level, step-wise injection is markedly more durable, giving a recipe for stable long-term agent memory.
- **Theme:** Memory for LLMs and agents
- **Relevance:** Directly attacks the durability problem in long-term/episodic agent memory, with concrete prescriptions for how to encode and inject experience.
- **Link:** https://arxiv.org/abs/2606.04703

## 2. M^3Eval: Multi-Modal Memory Evaluation through Cognitively-Grounded Video Tasks
- **Authors:** Jie Huang, Ruixun Liu, Sirui Sun, et al.
- **Summary:** M^3Eval is a cognitively-grounded benchmark that probes distinct memory dimensions (binding, disentanglement, symbolic recall) in multimodal models via video tasks. Across frontier MLLMs, the authors find consistent weaknesses in maintaining disentangled representations and symbolic memory, exposing failure modes that aggregate accuracy scores hide.
- **Theme:** Memory for LLMs and agents
- **Relevance:** A new diagnostic for memory in multimodal LLMs — useful both as an evaluation target and as a framing of what "memory" decomposes into.
- **Link:** https://arxiv.org/abs/2606.05008

## 3. Personal AI Agent for Camera Roll VQA
- **Authors:** Thao Nguyen, Krishna Kumar Singh, Donghyun Kim, et al.
- **Summary:** The paper introduces CamRoll, a dataset and conversational agent for VQA over a user's personal photo library, and argues that personalized visual memory cannot be handled with off-the-shelf long-context textual memory. They demonstrate retrieval and consistency strategies tuned for visual-personal context and report gains in answer faithfulness on user-specific queries.
- **Theme:** Memory for LLMs and agents
- **Relevance:** Tackles personalized, multimodal long-term memory — an under-explored slice that complements text-heavy retrieval-augmented memory work.
- **Link:** https://arxiv.org/abs/2606.05275

## 4. Streaming Communication in Multi-Agent Reasoning
- **Authors:** Zhen Yang, Xiaogang Xu, Wen Wang, et al.
- **Summary:** Instead of having agents exchange completed messages, this work pipelines intermediate reasoning tokens across agents so downstream agents can start consuming early, reliable steps. The authors derive step-level scaling laws for this streaming protocol and show meaningful latency reductions while preserving or improving final-answer quality on multi-agent reasoning benchmarks.
- **Theme:** Multi-agent frameworks & orchestration
- **Relevance:** A novel communication protocol for multi-agent reasoning — exactly the orchestration/communication primitive the theme targets, with empirical scaling analysis rather than just a framework demo.
- **Link:** https://arxiv.org/abs/2606.05158

## 5. Unsupervised Skill Discovery for Agentic Data Analysis
- **Authors:** Zhisong Qiu, Kangqi Song, Sheng Xia, et al.
- **Summary:** DataCOPE discovers reusable data-analysis skills for an agent through unsupervised exploration, paired with verifier-guided self-evaluation (adaptive checklists for report generation, agreement verifiers for reasoning). On data-analysis benchmarks the discovered skill library yields 9.71%–32.30% improvements without manual skill engineering.
- **Theme:** Multi-agent frameworks & orchestration
- **Relevance:** Skill libraries are a core orchestration primitive for tool-using agents, and the unsupervised + verifier-grounded recipe is more concrete than typical skill-acquisition proposals.
- **Link:** https://arxiv.org/abs/2606.06416

## 6. AdaPlanBench: Evaluating Adaptive Planning in Large Language Model Agents under World and User Constraints
- **Authors:** Jiayu Liu, Cheng Qian, Zhenhailong Wang, et al.
- **Summary:** AdaPlanBench is a multi-turn benchmark where world and user constraints are revealed progressively, forcing agents to revise plans on the fly. The strongest frontier model only reaches 67.75% accuracy, and the authors show monotonic degradation as constraints stack — quantifying how brittle current "adaptive" planning really is.
- **Theme:** Core LLM advances (planning / agentic evaluation)
- **Relevance:** A genuinely adversarial planning evaluation that targets the gap between static planning benchmarks and agent deployments where constraints shift mid-task.
- **Link:** https://arxiv.org/abs/2606.05622

## 7. AutoLab: Can Frontier Models Solve Long-Horizon Auto Research and Engineering Tasks?
- **Authors:** Zhangchen Xu, Junda Chen, Yue Huang, et al.
- **Summary:** AutoLab benchmarks frontier models on ultra-long-horizon closed-loop research and engineering tasks. The headline finding is that persistent iteration and empirical-feedback incorporation matter more than initial solution quality — agents that maintain and exploit a working state across many steps outperform stronger-but-stateless baselines.
- **Theme:** Core LLM advances (agentic-system evaluation)
- **Relevance:** Sharpens what "long-horizon agent" means empirically and surfaces state/feedback handling as the dominant factor — useful framing for both memory and orchestration work.
- **Link:** https://arxiv.org/abs/2606.05080

## 8. TIDE: Proactive Multi-Problem Discovery via Template-Guided Iteration
- **Authors:** Soyeong Jeong, Jinheon Baek, Minki Kang, Sung Ju Hwang
- **Summary:** TIDE attacks the under-studied problem of *discovering* multiple latent issues in a document or codebase rather than answering a single question. It iteratively surfaces small candidate batches conditioned on what has already been found, using thought templates to steer exploration, and improves coverage on both document and code benchmarks.
- **Theme:** Core LLM advances (reasoning / agent task design)
- **Relevance:** A clean methodological contribution to proactive agent behavior — a useful primitive for code-review and audit-style agents.
- **Link:** https://arxiv.org/abs/2606.04743

## 9. Reinforcement Learning Elicits Contextual Learning of Unseen Language Translation
- **Authors:** Hanxu Hu, Zdeněk Šnajdr, Pinzhen Chen, Jannis Vamvas, Rico Sennrich
- **Summary:** The authors RL-train translation models with a chrF reward and show that this elicits a meta-skill of consuming in-context linguistic resources (dictionaries, examples) rather than memorizing language pairs. On strictly unseen languages it beats supervised fine-tuning, suggesting RL is a more effective driver of in-context compositional skills than SFT.
- **Theme:** Core LLM advances (fine-tuning)
- **Relevance:** A clear, falsifiable claim that RL produces qualitatively different (more contextual) skills than SFT — relevant to anyone fine-tuning agents that must use retrieved context.
- **Link:** https://arxiv.org/abs/2606.06428

## 10. ArcANE: Do Role-Playing Language Agents Stay in Character at the Right Time?
- **Authors:** Woojung Song, Nalim Kim, Sangjun Song, et al.
- **Summary:** ArcANE evaluates whether role-playing LLM agents track *character arcs* — psychological evolution across a narrative — rather than holding a fixed persona. Conditioning generations on the explicit character arc outperforms persona-only and chronology-only baselines across 17 novels, especially in scenarios the source text never covered.
- **Theme:** Core LLM advances (agentic-system evaluation)
- **Relevance:** Reframes persona consistency as trajectory-following, which connects naturally to long-horizon agent memory and dynamic state tracking.
- **Link:** https://arxiv.org/abs/2606.05553
