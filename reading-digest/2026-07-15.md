## 1. LightMem-Ego: Your AI Memory for Everyday Life
- **Authors:** Yijun Chen, Boyi Xiao, Yixian Zhao, et al.
- **Summary:** Introduces a streaming multimodal memory system for wearable/mobile AI assistants that ingests continuous egocentric video + audio, aligns them on a shared timeline, and stores them in a three-tier hierarchy of current, short-term, and long-term memory. A query router chooses the appropriate tier and retrieves multimodal evidence to ground the answer, with a phone/AI-glasses deployment supporting tasks like object finding, conversation recall, and routine discovery.
- **Theme:** Memory for LLMs and agents (long-term/episodic multimodal memory, retrieval routing).
- **Relevance:** Direct hit on the memory theme — a concrete architecture for continuous accumulation, hierarchical organization, and multimodal retrieval on-device.
- **Link:** https://arxiv.org/abs/2607.11487

## 2. Multi-Agent LLMs Fail to Explore Each Other
- **Authors:** Hyeong Kyu Choi, Jiatong Li, Wendi Li, et al.
- **Summary:** Formalizes multi-agent exploration among LLM agents as a partially-observable stochastic game (POSG) in which each agent must probe peers to infer capabilities, and shows that current LLMs are myopic and polarize, hurting coordination. Proposes MACE (Multi-Agent Contextual Exploration), a lightweight framework that structures peer selection to promote exploration, and demonstrates gains across contextual and parametric diversity settings, with a theoretical result that exploration value grows with agent diversity.
- **Theme:** Multi-agent frameworks and orchestration (coordination, peer discovery).
- **Relevance:** Direct hit — diagnoses a systemic failure mode in multi-agent LLM setups and offers a general orchestration remedy.
- **Link:** https://arxiv.org/abs/2607.11250

## 3. Know Before Fix: QA-Driven Repository Knowledge Acquisition for Software Issue Resolution
- **Authors:** Haotian Lin, Silin Chen, Xiaodong Gu, et al.
- **Summary:** Proposes ACQUIRE, an agentic framework that separates repo knowledge acquisition from patch generation for automated issue resolution. A Questioner-Answerer pair produces structured repository knowledge for a downstream Resolver agent, yielding up to 4.4 pp Pass@1 gains on SWE-bench Verified over pre-repair baselines.
- **Theme:** Multi-agent frameworks and orchestration (role assignment: Questioner/Answerer/Resolver); relates to shared working memory for agents.
- **Relevance:** Concrete role-split multi-agent architecture with measurable code-agent gains.
- **Link:** https://arxiv.org/abs/2607.11111

## 4. Metacognition in LLMs: Foundations, Progress, and Opportunities
- **Authors:** Gabrielle Kaili-May Liu, Areeb Gani, Jacqueline Lu, et al.
- **Summary:** A comprehensive taxonomy of the emerging LLM-metacognition literature covering methods and benchmarks for measuring metacognitive abilities, techniques to elicit and improve them, and their implications for reliability. Consolidates approaches for self-monitoring, calibration, and self-directed learning into a shared vocabulary and roadmap.
- **Theme:** Core LLM advances (reasoning, planning, agentic-system evaluation).
- **Relevance:** Establishes a framework for reasoning-about-reasoning that is foundational to agentic systems and self-improvement loops.
- **Link:** https://arxiv.org/abs/2607.11881

## 5. Proxy Exploration and Reusable Guidance: A Modular LLM Post-Training Paradigm via Proxy-Guided Update Signals
- **Authors:** Daocheng Fu, Rong Wu, Yu Yang, et al.
- **Summary:** Introduces PUST (Proxy-guided Update Signal Transfer), which runs RL exploration on a small proxy model and transfers only the relative improvement signal to a stronger primary model, decoupling exploration from alignment. Experiments on Qwen3 across math and code show that signals extracted from much weaker proxies robustly lift stronger targets and can be cached, reused, and composed across models.
- **Theme:** Core LLM advances (fine-tuning / post-training / PEFT-adjacent).
- **Relevance:** Practical route to cheaper, modular RL post-training with weak-to-strong transfer — high leverage for downstream agent training pipelines.
- **Link:** https://arxiv.org/abs/2607.11505

## 6. AdvancedMathBench: A Benchmark Suite for Advanced Mathematical Proof Generation and Verification
- **Authors:** Lingkai Kong, Zijian Wu, Yuzhe Gu, et al.
- **Summary:** Releases ProverBench (296 undergraduate-through-qualifying-exam problems) plus VerifierBench (888 expert-annotated proof trajectories) for measuring both proof generation and proof-critique ability, together with an automatic verification pipeline that produces fine-grained error assessments. Even frontier models (GPT-5.5-xhigh) score only 75.8/66.1 on the UGD/QE splits and just 65.1 Balanced F1 on verification, identifying critical-error detection as a major bottleneck.
- **Theme:** Core LLM advances (reasoning, agentic-system evaluation).
- **Relevance:** New rigorous evaluation for reasoning that separates generation from verification — useful for benchmarking reasoning agents.
- **Link:** https://arxiv.org/abs/2607.11849

## 7. MET: Theory-Grounded and Culture-Aware Multilingual Moral Reasoning
- **Authors:** Ayoung Lee, Ryan Kwon, Yunxiang Zhang, et al.
- **Summary:** Contributes MCLASH (a culturally-adapted multilingual moral-reasoning benchmark), MET (a two-step prompting method that first selects culture/situation-specific moral grounds and then reasons in the user's native language), and MET-D, a self-distillation training stage that needs no external supervision. On Qwen3 and Gemma3 backbones MET-D yields ~4 macro-F1 gain on MCLASH and boosts native-language reasoning by ~62 points on average, with peaks up to +12.94 for Malay.
- **Theme:** Core LLM advances (reasoning method, self-distillation fine-tuning).
- **Relevance:** Combines a theory-grounded reasoning scaffold with a lightweight, supervision-free training loop — a reusable recipe beyond the moral domain.
- **Link:** https://arxiv.org/abs/2607.11736

## 8. Read It Back: Pretrained MLLMs Are Zero-Shot Reward Models for Text-to-Image Generation
- **Authors:** Runhui Huang, Qihui Zhang, Zhe Liu, et al.
- **Summary:** Proposes SpectraReward, a training-free reward function that repurposes pretrained multimodal LLMs as zero-shot judges by scoring how well the original prompt can be recovered from a generated image, plus a Self-SpectraReward variant for unified multimodal models. Consistent gains across diffusion backbones, RL algorithms, and MLLM judges surface reward-policy alignment as the key lever.
- **Theme:** Core LLM advances (reward modeling / RL post-training, adjacent to fine-tuning).
- **Relevance:** A general zero-shot-reward pattern (invert the target and score reconstruction) that transfers beyond image generation to any generative RL loop needing a cheap MLLM judge.
- **Link:** https://arxiv.org/abs/2607.11886
