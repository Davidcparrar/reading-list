## 1. EvolvingWorld: An Open-Schema Framework for Co-Evolving Role-Play Agents and World Model

- **Authors:** Qing Zong, Yue Guo, Mengxin Yang, et al.
- **Summary:** Proposes a framework where multiple role-play character agents and an LLM-based world model co-evolve during interactive literary simulations. Character agents keep persistent, open-schema profiles that are updated as events unfold, while the world model tracks the shared state so that long-horizon storylines remain coherent instead of drifting. The paper shows that this joint state maintenance produces more consistent characters and richer world dynamics than static role-play baselines.
- **Theme:** Memory (also relevant to multi-agent role-play).
- **Relevance note:** Directly targets persistent, evolving state — both per-agent memory and shared world memory — which is the intersection of themes (1) and (3).
- **Link:** https://arxiv.org/abs/2607.17250

## 2. SWE-Pruner Pro: Context Pruning for Coding Agents via Internal Representations

- **Authors:** Yuhang Wang, Yuling Shi, Shaoqiu Zhang, et al.
- **Summary:** Introduces a context-management layer for coding agents that inspects the model's internal representations to decide which prior tokens can be safely dropped from the working context. The pruner is trained to preserve information the agent still relies on for its next action, yielding up to 39% token savings while improving SWE-Bench task success by 3.8 points. The result is a concrete demonstration that learned working-memory compression can beat naive truncation without hurting downstream agentic performance.
- **Theme:** Memory (working / context memory for agents).
- **Relevance note:** A rare paper that treats the LLM's context window as a memory system to be actively managed, with an agentic evaluation on SWE-Bench.
- **Link:** https://arxiv.org/abs/2607.18213

## 3. RynnBrain 1.1: Embodied Foundation Models for Perception, Spatial Reasoning, and Planning

- **Authors:** Kehan Li, Bohan Hou, Minghao Zhu, et al.
- **Summary:** Releases a family of embodied foundation models spanning 2B to 122B parameters that unify perception, spatial reasoning, and multi-step planning behind a single interface, and reports deployment on physical robots. The scaling story focuses on how larger backbones translate into stronger planning and multimodal integration for agent stacks. Positioned as a foundation-tier model for downstream agentic pipelines rather than a task-specific system.
- **Theme:** Core LLM advances (reasoning, planning, agentic-system evaluation).
- **Relevance note:** A frontier planning-oriented model with real-world agentic evaluation; useful as a substrate to compare orchestration and memory ideas against.
- **Link:** https://arxiv.org/abs/2607.17977

## 4. LLM-as-a-Coach: Turning Feedback Models into Dense Supervisors for Open-Ended RL

- **Authors:** Tianzhu Ye, Li Dong, Guanheng Chen, et al.
- **Summary:** Reframes reward / feedback models as "coaches" that emit dense, token-level supervision signals instead of a single scalar reward, and shows how to plug those signals into a reinforcement-learning post-training loop. The dense signal preserves fine-grained preferences that scalar rewards flatten, and the paper reports gains on open-ended tasks where reward sparsity usually hurts credit assignment. The recipe is generic enough to sit on top of existing RLHF/RLAIF stacks.
- **Theme:** Core LLM advances (fine-tuning / RL post-training).
- **Relevance note:** Direct improvement to the RL fine-tuning stage that most agent systems now depend on.
- **Link:** https://arxiv.org/abs/2607.18110

## 5. Distilled Reinforcement Learning: Combining Teacher Supervision with RL Objectives

- **Authors:** Chen Wang, Zhaochun Li, Jionghao Bai, et al.
- **Summary:** Integrates teacher-model distillation with an on-policy RL loss, using importance sampling and a geometric normalization step to keep the two signals compatible when the student and teacher live in different tokenizer / policy geometries. The method transfers reasoning ability across model families more reliably than either pure distillation or pure RL, and reports gains on standard reasoning benchmarks. Useful when a smaller student needs to inherit a specific reasoning style from a larger, potentially closed teacher.
- **Theme:** Core LLM advances (fine-tuning, reasoning transfer).
- **Relevance note:** A practical, cross-family recipe for RL-flavored distillation — a common bottleneck when trying to reproduce frontier reasoning behavior in smaller open models.
- **Link:** https://arxiv.org/abs/2607.17247

## 6. The Geometry of Semantic Space: A Continuous Framework for the Transformer Architecture

- **Authors:** Zhihua Liang
- **Summary:** Recasts Transformer forward passes as integro-differential equations acting on a continuous semantic manifold, and derives predictions about attention concentration, optimization dynamics, and a phase transition at the effective context limit. The framework is theoretical rather than empirical, but yields testable claims — for example about when in-context learning should abruptly degrade — that can be checked against existing models. Positions itself as a unifying mathematical picture rather than a new architecture.
- **Theme:** Core LLM advances (theoretical foundations of reasoning / context behavior).
- **Relevance note:** Included as a theoretical anchor for the reasoning / context-length behavior that memory and agent papers implicitly rely on; lower-priority than the applied entries above.
- **Link:** https://arxiv.org/abs/2607.17146
