## 1. LightMem-Ego: Your AI Memory for Everyday Life
- **Authors:** Yijun Chen, Boyi Xiao, Yixian Zhao, et al.
- **Summary:** Introduces a lightweight, streaming multimodal memory system that continuously ingests egocentric audio+video and organizes them along a shared timeline into a three-tier hierarchy (current, short-term, long-term). At query time the system dynamically routes retrieval to the appropriate tier and answers with grounded multimodal evidence, and the authors demonstrate deployment on smartphones and AI glasses for tasks like conversation recall, routine discovery, and personalized assistance.
- **Theme:** Memory for LLMs and agents
- **Relevance:** A concrete, deployed memory-hierarchy design that directly targets the long-term/episodic/working-memory split we care about, with an on-device retrieval router as the key contribution.
- **Link:** https://arxiv.org/abs/2607.11487

## 2. Extending LLM Context via Associative Recurrent Memory
- **Authors:** Gleb Kuzmin, Ivan Rodkin, Aydar Bulatov, et al.
- **Summary:** Proposes an Associative Recurrent Memory Transformer (ARMT) that augments a base LLM with a compact associative memory carried across chunks, allowing effective context to grow without a corresponding blow-up in KV-cache size. The design keeps memory cost roughly constant while extending usable context well past the pretrained window, with reported gains on long-context evaluations.
- **Theme:** Memory for LLMs and agents
- **Relevance:** A parameter-efficient recurrent-memory approach to long-context handling — sits squarely at the intersection of working memory and context/state management.
- **Link:** https://arxiv.org/abs/2607.11614

## 3. RAGU: A Multi-Step GraphRAG Engine with a Compact Domain-Adapted LLM
- **Authors:** Mikhail Komarov, Ivan Bondarenko, Stanislav Shtuka, et al.
- **Summary:** Presents a graph-based retrieval-augmented pipeline that separates entity/relation extraction from consolidation, using two-stage typed extraction, DBSCAN-backed deduplication, LLM-driven summarization, and Leiden community detection to build the knowledge graph. A compact domain-adapted LLM drives the multi-step retrieval and reasoning, giving a practical GraphRAG stack aimed at retrieval-augmented memory in narrow domains.
- **Theme:** Memory for LLMs and agents
- **Relevance:** Retrieval-augmented long-term memory built on an explicit graph store — a useful reference architecture for agentic memory pipelines beyond flat vector RAG.
- **Link:** https://arxiv.org/abs/2607.11683

## 4. Forgetting Our Way to Shared Meaning: Effects of Forgetting on Conceptual Alignment in a Non-Partnership Coordination Game
- **Authors:** Landon Liu, Mary Kelly, Alan Tsang
- **Summary:** Models a multi-agent coordination game where partners are re-shuffled every round and studies how different forgetting profiles in each agent's memory affect the emergence of a shared conceptual vocabulary. The authors show that bounded, structured forgetting can actually accelerate alignment across the population versus perfect recall, arguing that memory decay is a coordination mechanism, not just a limitation.
- **Theme:** Multi-agent frameworks & orchestration (with a memory angle)
- **Relevance:** Bridges memory and multi-agent themes: an explicit study of how agent memory characteristics shape emergent multi-agent behaviour.
- **Link:** https://arxiv.org/abs/2607.11787

## 5. StructAgent: Harness Long-horizon Digital Agents with Unified Causal Structure
- **Authors:** StructAgent authors, et al.
- **Summary:** Proposes an agent harness that maintains a unified, compact "state" object capturing verifiable task progress, coupled with a structured workflow that regulates transitions through verifier-backed checks. The goal is to keep long-horizon digital agents grounded in an auditable causal structure rather than free-form scratchpad state, and the authors report improved reliability on multi-step tasks.
- **Theme:** Multi-agent frameworks & orchestration
- **Relevance:** A structured orchestration/state-management design for long-horizon digital agents — directly relevant to agent architecture and control flow.
- **Link:** https://arxiv.org/abs/2607.11388

## 6. When Local Monitors Miss Compositional Harm: Diagnosing Distributed Backdoors in Multi-Agent Systems
- **Authors:** Yibo Hu, Ren Wang
- **Summary:** Studies a threat model where a harmful payload is decomposed into sub-behaviours across multiple cooperating agents, so each per-agent monitor sees only benign fragments while the assembled multi-agent output is malicious. The paper formalises "distributed backdoors" for multi-agent LLM systems, gives concrete attack constructions, and shows local safety filters systematically miss them — motivating cross-agent, compositional monitoring.
- **Theme:** Multi-agent frameworks & orchestration
- **Relevance:** Highlights a coordination/communication failure mode unique to multi-agent orchestration; important context when designing agent protocols and safety layers.
- **Link:** https://arxiv.org/abs/2607.11751

## 7. GEIS: A Generation-Evaluation-Improvement Loop of Agent Skills for Long-Form Article Generation
- **Authors:** Jiale Zhang, Juntao Hu, Zhijian Ou
- **Summary:** Structures long-form article generation as a loop over declaratively named "skills" (outlining, drafting, fact-checking, revising), with each iteration running generation, evaluation, and targeted improvement steps. The framework composes multiple specialised agents/skills into a controllable pipeline and shows better factuality and structure than monolithic single-pass generation on Wikipedia-style targets.
- **Theme:** Multi-agent frameworks & orchestration
- **Relevance:** A concrete role/skill assignment pattern for multi-agent orchestration on a long-horizon writing task, with a built-in evaluation loop.
- **Link:** https://arxiv.org/abs/2607.11503

## 8. Agentic Routing: The Harness-Native Data Flywheel
- **Authors:** Xinchen Liu, Hang Zhou, Yingjie Zong, et al.
- **Summary:** Proposes step-level routing inside an agent harness: the harness picks which underlying model to invoke at each execution step based on the current state, rather than committing to a single model for the whole trajectory. The routing decisions themselves generate labelled training data ("harness-native flywheel"), so the router progressively improves as the agent runs.
- **Theme:** Multi-agent frameworks & orchestration
- **Relevance:** Directly addresses tool/model assignment within agent harnesses and provides a self-improving orchestration signal — useful for multi-model agent stacks.
- **Link:** https://arxiv.org/abs/2607.11399

## 9. AdvancedMathBench: A Benchmark Suite for Advanced Mathematical Proof Generation and Verification
- **Authors:** Lingkai Kong, Zijian Wu, Yuzhe Gu, et al.
- **Summary:** Releases ProverBench (296 undergrad/qualifying-exam-level proof problems) and VerifierBench (888 model-generated proof trajectories with expert ground truth), targeting both proof generation and step-level proof verification. A dedicated automatic verifier trained on expert annotations delivers correctness verdicts plus fine-grained error labels; even the strongest models tested reach only ~76% on generation and ~65 balanced F1 on verification, exposing large headroom.
- **Theme:** Core LLM advances (agentic-system evaluation / reasoning)
- **Relevance:** Rigorous, process-level evaluation for advanced mathematical reasoning — the kind of benchmark that stresses planning and verification, not just final-answer heuristics.
- **Link:** https://arxiv.org/abs/2607.11849

## 10. Metacognition in LLMs: Foundations, Progress, and Opportunities
- **Authors:** Gabrielle Kaili-May Liu, Areeb Gani, Jacqueline Lu, et al.
- **Summary:** Surveys the emerging area of LLM metacognition, laying out a taxonomy for measurement, elicitation, and application of self-monitoring and self-regulation in language models. The authors organise benchmarks and methods for calibrated uncertainty, self-critique, and adaptive strategy selection, and highlight open questions around evaluation and reliability for agentic settings.
- **Theme:** Core LLM advances (reasoning)
- **Relevance:** A structured map of a fast-moving area central to agent reliability — useful as a reference when designing self-critique and confidence-aware reasoning loops.
- **Link:** https://arxiv.org/abs/2607.11881
