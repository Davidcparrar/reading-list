# Reading digest — 2026-06-04

Found 10 paper(s) in the last 48h window.

## 1. MemTrain: Self-Supervised Context Memory Training
- Authors: Ziheng Li, Xingrun Xing, Haoqing Wang et al.
- Summary: Proposes a self-supervised pre-training stage that targets the context-memory capability of LLM agents using two coupled proxy tasks on unlabeled Wikipedia: an end-to-end masked-entity reconstruction objective across multiple memory-update rounds, and an intermediate memory-recall objective that forces faithful compression of historical state. Both objectives are jointly optimized with GRPO, and downstream post-training on long-text QA and search QA improves by up to 17.67 points over direct task-specific fine-tuning.
- Theme: Memory for LLMs and agents
- Relevance note: Directly attacks the bottleneck of building general context-memory behavior without expensive annotated memory tasks — exactly the kind of memory-mechanism work the digest prioritizes.
- Link: https://arxiv.org/abs/2606.03197

## 2. Language Models Need Sleep: Learning to Self-Modify and Consolidate Memories
- Authors: Ali Behrouz, Farnoosh Hashemi, Vahab Mirrokni
- Summary: Introduces a biologically-inspired "Sleep" paradigm for continual learning in LLMs, with two phases: a Memory Consolidation stage (Knowledge Seeding) that distills a smaller-self's transient memories into a larger network by combining on-policy distillation with RL-based imitation, and a Dreaming stage in which the model generates a synthetic-data curriculum to rehearse and refine its capabilities without human supervision. The work positions short-term in-context state and long-term parametric memory as two ends of a continual-learning loop the model itself drives.
- Theme: Memory for LLMs and agents
- Relevance note: Reframes memory consolidation as a first-class training stage with concrete RL-distillation machinery, from a Google team (Mirrokni) — high-signal new direction for long-term LLM memory.
- Link: https://arxiv.org/abs/2606.03979

## 3. Streaming Communication in Multi-Agent Reasoning
- Authors: Zhen Yang, Xiaogang Xu, Wen Wang
- Summary: StreamMA replaces the standard "generate-then-transfer" pattern in multi-agent pipelines with token-level streaming, so each reasoning step is forwarded to downstream agents as soon as it is produced, pipelining adjacent agents to cut end-to-end latency. The authors give what they describe as the first closed-form joint analysis comparing streaming, serial, and single-agent protocols, and report average +7.3 pp gains (up to +22.4 pp on HMMT 2026) across Chain, Tree, and Graph topologies with Claude Opus 4.6 and GPT-5.4 — they also observe a "step-level scaling law" orthogonal to agent-count scaling.
- Theme: Multi-agent frameworks & orchestration
- Relevance note: Concrete communication protocol for multi-agent systems with both theoretical and empirical results across topologies — top match for the multi-agent theme.
- Link: https://arxiv.org/abs/2606.05158

## 4. M^3Eval: Multi-Modal Memory Evaluation through Cognitively-Grounded Video Tasks
- Authors: Jie Huang, Ruixun Liu, Sirui Sun
- Summary: Builds a cognitive-psychology-grounded benchmark that isolates distinct memory dimensions in multimodal models — retention, fidelity, robustness under interference, parallel-stream tracking, and symbolic vs. spatial vs. temporal grounding — rather than conflating memory with general video reasoning. Across representative MLLMs the benchmark surfaces consistent failure modes (e.g., interference patterns unlike humans, weak temporal grounding, weak symbolic memory) that current video benchmarks miss.
- Theme: Memory for LLMs and agents
- Relevance note: First systematic memory-isolating benchmark for multimodal models; useful diagnostic for any LLM/agent memory research.
- Link: https://arxiv.org/abs/2606.05008

## 5. Echo-Infinity: Learning Evolving Memory for Real-Time Infinite Video Generation
- Authors: Yuxuan Bian, Zeyue Xue, Songchun Zhang
- Summary: Replaces handcrafted KV-cache schedules and fixed-ratio history compression with a learnable "Memory Query" that is updated by attention and a gating mechanism whenever frames are evicted from the local window, producing an evolving memory state trained end-to-end with video diffusion transformers. A Unified Relative RoPE Recipe anchors sink frames and bounds the newest-frame index inside the pretrained temporal-RoPE range, closing the train-test extrapolation gap and supporting reported 24-hour real-time rollouts.
- Theme: Memory for LLMs and agents
- Relevance note: Concrete learnable long-horizon memory mechanism with constant-cost compression; the mechanism (gated, attention-updated memory queries) is portable beyond video generation.
- Link: https://arxiv.org/abs/2606.04527

## 6. World Models Meet Language Models: On the Complementarity of Concrete and Abstract Reasoning
- Authors: Yucheng Zhou, Wei Tao, Yiwen Guo et al.
- Summary: Frames the integration of stochastic video world-model rollouts with MLLM abstract reasoning as "controlled concrete reasoning," where the policy must decide when to invoke visual simulation, judge rollout credibility, and integrate it with abstract reasoning. They introduce VRQABench and OpenWorldQA, plus PF-OPSD — a training method that uses ground-truth future videos only as teacher-side privileged context to grade on-policy student trajectories — and report +10.6 / +10.9 pp gains and better robustness to noisy rollouts.
- Theme: Core LLM advances (reasoning, planning)
- Relevance note: Novel privileged-future self-distillation recipe for reasoning under uncertain simulation — directly relevant to planning agents that mix model-based imagination with LLM reasoning.
- Link: https://arxiv.org/abs/2606.03603

## 7. ThoughtFold: Folding Reasoning Chains via Introspective Preference Learning
- Authors: Ziyan Liu, Xueda Shen, Yuzhe Gu
- Summary: Argues that outcome-based RLVR reinforces the redundant trial-and-error inside correct long CoTs and proposes an introspective pipeline that identifies a spectrum of candidate sub-trajectories within each correct trace. A masked preference-optimization objective then explicitly penalizes the redundant segments and rewards direct bridging of essential reasoning steps, reportedly cutting DeepSeek-R1-Distill-Qwen-7B token usage by ~56% while preserving state-of-the-art accuracy.
- Theme: Core LLM advances (reasoning, planning)
- Relevance note: Fine-grained preference signal targeting overthinking, with a clean efficiency-vs-accuracy result; the introspective sub-trajectory construction is a reusable training-data recipe.
- Link: https://arxiv.org/abs/2606.03503

## 8. KVarN: Variance-Normalized KV-Cache Quantization Mitigates Error Accumulation in Reasoning Tasks
- Authors: Lorenz K. Muller, Philippe Bich, Chiara Boretti et al.
- Summary: Shows that under long-horizon autoregressive decoding KV-cache quantization errors accumulate across timesteps and are driven mostly by mis-scaled outlier tokens — a regime that prefill-style evaluations hide. KVarN is calibration-free: it applies a Hadamard rotation followed by dual-axis variance normalization of K and V, setting new state-of-the-art at 2-bit precision on MATH500, AIME24, and HumanEval.
- Theme: Core LLM advances (reasoning-time efficiency)
- Relevance note: Memory-system advance that directly enables longer reasoning under tight KV budgets — useful primitive for any memory-extended LLM system.
- Link: https://arxiv.org/abs/2606.03458

## 9. Reproducing, Analyzing, and Detecting Reward Hacking in Rubric-Based Reinforcement Learning
- Authors: Xuekang Wang, Zhuoyuan Hao, Shuo Hou
- Summary: Introduces CHERRL, a controllable environment that injects known biases into an LLM-as-Judge so that rubric-based RL reward hacking can be reproduced cleanly, observed as a divergence between true reward and judge reward, and timestamped at onset. The authors then study judge biases by discoverability and exploitability and prototype an agent-based detector that flags reward-hacking onset directly from training logs.
- Theme: Multi-agent frameworks & orchestration (agentic-system evaluation)
- Relevance note: Practical testbed plus an agentic detection loop for a failure mode that increasingly matters as agentic RL training pipelines proliferate.
- Link: https://arxiv.org/abs/2606.04923

## 10. Self-Distilled Policy Gradient (SDPG)
- Authors: Yifeng Liu, Shiyuan Zhang, Yifan Zhang
- Summary: Casts on-policy self-distillation as an auxiliary full-vocabulary student-to-teacher reverse-KL loss and combines it with group-relative verifier advantages (normalized by standard deviation) and reference-policy KL regularization, yielding a unified self-distilled policy-gradient recipe for sparse-reward RL. Reported gains in stability and final performance over both RLVR and pure self-distillation baselines on reasoning tasks.
- Theme: Core LLM advances (fine-tuning / agentic post-training)
- Relevance note: Clean RL-with-self-distillation recipe directly relevant to post-training LLM agents under sparse rewards.
- Link: https://arxiv.org/abs/2606.04036
