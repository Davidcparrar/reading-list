1. **Supersede: Diagnosing and Training the Memory-Update Gap in LLM Agents**
   - Authors: Vedant Patel
   - Summary: The paper isolates a specific failure mode in long-running LLM agents — the inability to overwrite a previously-stored fact when reality changes — and shows that replacing an agent's full conversation context with a bounded, self-maintained memory drops accuracy from 92% to 77%, with the gap widening as fact supersession events accumulate. It releases an open RL environment named Supersede that scores how well an agent rewrites stale entries, and reports that GRPO fine-tuning on this environment roughly doubles memory-update accuracy without harming retrieval. The contribution is a clean, training-time diagnostic for the most under-measured axis of agent memory.
   - Theme: (1) memory for LLMs and agents.
   - Relevance note: Targets the exact crack in agent memory — write-after-write — that prior memory papers gloss over with retrieval scores.
   - Link: https://arxiv.org/abs/2606.27472

2. **GBC: Gradient-Based Connections for Optimizing Multi-Agent Systems**
   - Authors: Xiaocheng Yang, Abdulrahman Alrabah, Dilek Hakkani-Tür, et al.
   - Summary: GBC reframes a multi-agent LLM system as a computational graph and introduces gradient-flowing "connection weights" between agents that quantify, at the token level, how much each upstream agent's output influences a downstream agent's loss. Backpropagating a task loss through this attribution graph yields fine-grained credit assignment, which is then used to drive targeted prompt optimization for the agents that mattered. The released implementation (AgentChord) reports gains over strong single- and multi-agent baselines on MultiWOZ and τ-bench, with higher attribution quality correlating with larger optimization wins.
   - Theme: (2) multi-agent frameworks & orchestration.
   - Relevance note: First plausibly-general credit-assignment recipe for prompt-based MAS — directly addresses the "which agent broke this run?" question.
   - Link: https://arxiv.org/abs/2606.28187

3. **Temporal Validity in Retrieval Memory: Eliminating Stale-Fact Errors for AI Agents over Evolving Knowledge** (MemStrata)
   - Authors: Neeraj Yadav
   - Summary: MemStrata layers an explicit temporal-validity model over a retrieval memory: every stored fact carries effective/expiry timestamps and is checked against a deterministic supersession rule at query time, so stale entries cannot win a similarity match. The authors evaluate on evolving-knowledge benchmarks where facts mutate over the conversation and report 0.95–1.00 accuracy versus much lower numbers for vanilla retrieval. The pitch is that "freshness as a typed predicate" is the missing piece in most agent memory stacks.
   - Theme: (1) memory for LLMs and agents.
   - Relevance note: Complementary to Supersede — Supersede asks whether the agent re-writes, MemStrata asks whether the store can refuse the stale read.
   - Link: https://arxiv.org/abs/2606.26511

4. **DMV-Bench: Diagnosing Long-Horizon Multimodal Agents' Visual Memory with Incidental Cue Injection**
   - Authors: Yujin Tang, Chenming Shang, Ruize Xu, Nikhil Singh
   - Summary: DMV-Bench is the first interactive benchmark targeting visual memory in long-horizon multimodal agents: pre-rendered incidental cues are injected into extended shopping sessions, and the agent is later quizzed on whether it can recall what it actually saw rather than what it was told. Alongside the benchmark, the authors propose DualMem, which maintains parallel visual and verbal codes for the same observation, and report that it beats recent multimodal-agent memory systems at every chain length tested. Useful as both a measurement tool and a baseline.
   - Theme: (1) memory for LLMs and agents.
   - Relevance note: The visual side of agent memory is barely measured — this gives both a knob to turn and a baseline to beat.
   - Link: https://arxiv.org/abs/2606.27499

5. **QueenBee Planner: Skill-Evolving Communication Topologies for Token-Efficient LLM Multi-Agent Systems**
   - Authors: Congjia Tian, Yuhang Yao, Jiaming Cui
   - Summary: QueenBee treats inter-agent communication topology as itself a retrievable, self-improving skill: a planner emits a temporal DAG describing who-talks-to-whom-when, execution traces are distilled into evidence-backed rules (Preserve / Modify / Avoid), and the rules feed back to bias the next plan. The reported numbers show self-evolved graphs cutting RMSE from 12.53 to 7.87 while also reducing token spend versus full-broadcast baselines. A neat formalization of an idea that's usually left implicit in MAS papers.
   - Theme: (2) multi-agent frameworks & orchestration.
   - Relevance note: Communication topology is the single biggest cost lever in MAS; this paper makes it an explicit, learnable artifact.
   - Link: https://arxiv.org/abs/2606.27492

6. **Memory Depth, Not Memory Access: Selective Parametric Consolidation for Long-Running Language Agents**
   - Authors: Haoliang Han
   - Summary: The paper argues that the right axis for long-running agent memory isn't "can the agent retrieve the fact" but "has the agent internalized the fact in weights deeply enough to act on it goal-conditionally," and proposes selective parametric consolidation — pushing a small, carefully-chosen subset of episodic traces into LoRA-style parameter updates during idle time. The split between retrieval-time access and weight-time depth is operationalized so that the cost stays bounded as conversations get longer. Results focus on durable goal pursuit in long-horizon tasks.
   - Theme: (1) memory for LLMs and agents.
   - Relevance note: Counter-take to RAG-everywhere: argues that some agent memory has to be parametric or the agent will not actually behave as if it remembers.
   - Link: https://arxiv.org/abs/2606.26806

7. **Tandem Reinforcement Learning with Verifiable Rewards**
   - Authors: Difan Jiao, Raghav Singhal, Robert West, Ashton Anderson
   - Summary: Tandem RL trains a stronger "senior" agent to co-generate a reasoning trace alongside a frozen "junior" — the senior emits the chain-of-thought but is rewarded only when the junior can complete the trace and reach the verifiable answer. The result is a senior that matches solo capability while producing chains that hand off more cleanly: less distributional drift on the junior, less brittleness when the junior takes over mid-trace. Effectively a multi-agent-aware training objective for the chain-of-thought itself.
   - Theme: (2) multi-agent frameworks & orchestration (with a strong (3) core-LLM training angle).
   - Relevance note: Concrete training recipe that turns the strong-model / weak-model pairing into a first-class optimization target, not an inference-time pattern.
   - Link: https://arxiv.org/abs/2606.28166

8. **When Does Combining Language Models Help? A Co-Failure Ceiling on Routing, Voting, and Mixture-of-Agents Across 67 Frontier Models**
   - Authors: (first author not parsed from arXiv feed)
   - Summary: A large empirical study across 67 frontier LLMs derives a simple ceiling: the best achievable accuracy of any ensembling scheme (routing, voting, mixture-of-agents) cannot exceed 1 − β, where β is the rate at which every model is wrong on the same query. They then sweep popular schemes and find that without strong query-conditioned routing signals, "combining models rarely beats the single best model" on most benchmarks. A cold-water result that disciplines a lot of MAS work that quietly assumes ensembling is free upside.
   - Theme: (2) multi-agent frameworks & orchestration (negative result).
   - Relevance note: Useful baseline-of-baselines for any MAS claim — if a new framework cannot beat single-best, the co-failure ceiling tells you why.
   - Link: https://arxiv.org/abs/2606.27288

9. **Formalizing Latent Thoughts: Four Axioms of Thought Representation in LLMs**
   - Authors: Fahd Seddik, Fatemeh Fard
   - Summary: The paper proposes a benchmark-independent way to score the latent "thought" representations inside LLMs along four axioms — Causality, Minimality, Separability, Stability — each computed directly on the representation rather than on downstream accuracy. Auditing open-weight LLMs across 23 reasoning tasks, the authors find that no model satisfies all four axioms, the representations distinguish task type but not two questions within the same task, and they carry little information beyond what is already in the input embedding. The failure is consistent across dense, reasoning-distilled, and RL-trained families, pointing to a structural limit rather than a scale or training-procedure artifact.
   - Theme: (3) core LLM advances (reasoning / representation evaluation).
   - Relevance note: Gives a way to ask "is this model's reasoning real or is the benchmark covering for it" without writing a new task.
   - Link: https://arxiv.org/abs/2606.27378

10. **ATOD: Annealed Turn-aware On-policy Distillation for Multi-turn Autonomous Agents**
    - Authors: Qitai Tan, Zefang Zong, Yang Li, Peng Chen
    - Summary: ATOD combines on-policy distillation with reinforcement learning for multi-turn agents and adds a Turn-level Disagreement-Uncertainty Reweighting term that scales each turn's loss by how much the student and teacher disagree there. The annealing schedule walks the student off teacher mimicry and onto outcome reward as training progresses. Reported gains over plain OPD (+3.03) and over GRPO (+23.62) across ALFWorld, WebShop, and Search-QA, suggesting per-turn weighting matters more than the choice of base objective.
    - Theme: (3) core LLM advances (training recipe for agentic models).
    - Relevance note: Directly comparable to OPID (covered in the 2026-06-26 digest) — turn-aware loss weighting is the new knob.
    - Link: https://arxiv.org/abs/2606.27814
