# Reading digest — 2026-06-24

Found 10 paper(s) in the last 48h window.

1. **Escaping the Self-Confirmation Trap: An Execute-Distill-Verify Paradigm for Agentic Experience Learning**
   - Authors: Shiding Zhu, Yudi Qi, Yajie Wang, et al.
   - Summary: Single-agent experience-learning loops tend to canonize wrong-but-self-consistent trajectories as "successful" memories, so the authors split the loop across heterogeneous executor agents, a third-party distiller that compares trajectories, and a verifier consensus that gates what is written to shared or private memory. The method improves long-horizon performance on tau2-bench, Mind2Web and MMTB, showing that decoupling write-access to agent memory across roles is itself a useful inductive bias.
   - Theme: Memory for LLMs and agents (also multi-agent orchestration).
   - Relevance: Directly tackles how trajectories become long-term agent memory — central to the priority-1 theme and a clean recipe to copy.
   - Link: https://arxiv.org/abs/2606.24428

2. **Self-Compacting Language Model Agents**
   - Authors: Tianjian Li, Jingyu Zhang, William Jurayj, et al.
   - Summary: Instead of compacting an agent's trace at fixed token thresholds, the model is given a compaction tool and a short rubric telling it to fire only at sub-task boundaries or convergence and to suppress mid-derivation. The result is a context-management policy the agent learns to invoke itself, preserving partial results that interval-based scaffolds would shred.
   - Theme: Memory for LLMs and agents (context/state management).
   - Relevance: A concrete mechanism for working-memory hygiene in long agent runs — directly aligned with the priority-1 theme.
   - Link: https://arxiv.org/abs/2606.23525

3. **Unlimited OCR Works**
   - Authors: Youyang Yin, Huanhuan Liu, YY
   - Summary: Long end-to-end OCR runs balloon the KV cache, so the authors swap the decoder's attention for Reference Sliding Window Attention (R-SWA), a working-memory mechanism that streams pages without monotonically growing state. They report parity or gains over a DeepSeek OCR baseline with bounded memory, framing the design as emulating human "parsing working memory."
   - Theme: Memory for LLMs and agents (working-memory architecture).
   - Relevance: A bounded-memory attention variant motivated explicitly by working-memory principles — useful prior art for any long-context agent system.
   - Link: https://arxiv.org/abs/2606.23050

4. **DREAM: Dense Retrieval Embeddings via Autoregressive Modeling**
   - Authors: Yixuan Tang, Yi Yang
   - Summary: DREAM trains a dense retriever without contrastive labels by injecting retriever-produced query-document similarities into selected attention heads of a frozen LLM, so the LLM's next-token prediction loss on the conditioned answer flows back into the retriever via attention. The end-to-end signal lets next-token prediction substitute for hand-labeled positives/negatives in retrieval training.
   - Theme: Memory for LLMs and agents (retrieval-augmented memory).
   - Relevance: A label-free path to better retrievers — directly relevant for retrieval-augmented agent memory and the broader RAG stack.
   - Link: https://arxiv.org/abs/2606.24667

5. **Qwen-AgentWorld: Language World Models for General Agents**
   - Authors: Yuxin Zuo, Zikai Xiao, Li Sheng, et al.
   - Summary: The authors train language-world-model variants (Qwen-AgentWorld-35B-A3B and -397B-A17B) to simulate seven agentic environments by predicting next-state via long chain-of-thought, using a CPT→SFT→RL pipeline over 10M+ interaction trajectories. The result is a backbone that lets agents reason and plan inside a learned simulator rather than only in the deployed environment.
   - Theme: Multi-agent frameworks & orchestration (planning substrate for agents).
   - Relevance: A serious attempt at a foundation model that doubles as an environment simulator — relevant for agent planning and rollout-driven training.
   - Link: https://arxiv.org/abs/2606.24597

6. **AOHP: An Open-Source OS-Level Agent Harness for Personalized, Efficient and Secure Interaction**
   - Authors: Shanhui Zhao, Jiacheng Liu, Guohong Liu, et al.
   - Summary: AOHP treats AI agents as first-class actors inside the Android Open Source Project, exposing personalized service composition, agent-friendly UI surfaces, and a secure information-flow layer between agents and the OS. Their reported numbers show +21% task completion, −51% token cost, and better security-policy compliance versus conventional app-centric stacks.
   - Theme: Multi-agent frameworks & orchestration.
   - Relevance: A rare OS-level take on agent infrastructure — useful as architectural prior art for any agent that must coordinate across many apps.
   - Link: https://arxiv.org/abs/2606.23449

7. **ReMMD: Realistic Multilingual Multi-Image Agentic Verification for Multimodal Misinformation Detection**
   - Authors: Chenhao Dang, Dantong Zhu, Jun Yang, et al.
   - Summary: ReMMD-Agent decomposes a viral post into atomic claims, searches and persists multi-modal evidence in a shared memory, and predicts a structured L1/L2/L3 veracity label, all evaluated on a 500-sample multilingual multi-image benchmark. The persistent-memory verifier hits 41.8% five-way accuracy at substantially lower token cost than prior MMD-Agent and T2-Agent baselines.
   - Theme: Multi-agent frameworks & orchestration (also memory).
   - Relevance: A realistic case study of an agent that earns its keep through structured, persistent shared memory — relevant to both priority-1 and priority-2 themes.
   - Link: https://arxiv.org/abs/2606.24112

8. **EnterpriseClawBench: Benchmarking Agents from Real Workplace Sessions**
   - Authors: Jincheng Zhong, Weizhi Wang, Che Jiang, et al.
   - Summary: The authors mine 852 reproducible tasks from a large archive of real enterprise agent sessions, packaging each with fixtures, rewritten prompts, role/skill taxonomy, hard rules, and semantic rubrics; the benchmark data itself stays private, but the construction protocol is reusable. Even the best configuration (Codex + GPT-5.5) only reaches 0.663, suggesting current agents miss enterprise-style structure.
   - Theme: Multi-agent frameworks & orchestration (agentic-system evaluation).
   - Relevance: One of the more honest evaluations of agents inside an actual workplace workflow — useful for calibrating expectations of "agent products."
   - Link: https://arxiv.org/abs/2606.23654

9. **OpenThoughts-Agent: Data Recipes for Agentic Models**
   - Authors: Negin Raoof, Richard Zhuang, Marianna Nezhurina, et al.
   - Summary: OT-Agent runs 100+ controlled ablations across the agentic data-curation pipeline (task sources, diversity, format) and then trains Qwen3-32B on the resulting 100K mix, reaching 44.8% average across seven agentic benchmarks — a 3.9-point gain over the strongest prior open dataset (Nemotron-Terminal-32B at 40.9%). They also show favorable data-scaling curves versus competing open recipes.
   - Theme: Core LLM advances for agents (fine-tuning data curation).
   - Relevance: A rare open, ablation-driven recipe for training generalist agents — directly relevant to anyone training their own agentic model.
   - Link: https://arxiv.org/abs/2606.24855

10. **NatureBench: Can Coding Agents Match the Published SOTA of Nature-Family Papers?**
    - Authors: Yuru Wang, Lejun Cheng, Yuxin Zuo, et al.
    - Summary: NatureBench distills 90 cross-discipline tasks from Nature-family papers into per-task containerized environments via the NatureGym pipeline, then tests ten frontier agent configurations under a strict no-web-search protocol. The strongest configuration beats published SOTA on only 17.8% of tasks (g>0.1), and the failure mode is mostly wrong method choice and budget — not misunderstanding — i.e., agents translate problems into familiar supervised setups rather than inventing methods.
    - Theme: Core LLM advances for agents (agentic-system evaluation).
    - Relevance: A useful adversarial test of whether agents do science vs. just plumb existing recipes — relevant for calibrating coding-agent claims.
    - Link: https://arxiv.org/abs/2606.24530
