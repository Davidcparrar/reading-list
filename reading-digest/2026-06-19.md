# Reading digest — 2026-06-19

Found 10 paper(s) in the last 48h window.

## 1. Beyond the Current Observation: Evaluating Multimodal Large Language Models in Controllable Non-Markov Games
- **Authors:** Shengyuan Ding, Xilin Wei, Xinyu Fang, et al.
- **Summary:** Introduces RNG-Bench, a controllable benchmark that probes whether MLLMs can reconstruct past observations and act on them across long interactive episodes (Matching Pairs and a 3D Maze). The authors define a "Memory Gap" metric that disentangles forgetting from poor decision-making by comparing identical prompts with and without prior observations re-inserted, and they evaluate frontier MLLMs at contexts up to ~128K tokens with ~350 images per episode.
- **Theme:** Memory for LLMs and agents
- **Relevance:** Direct, controlled benchmark for episodic/working memory in agentic MLLMs with a dedicated memory metric — exactly the kind of evaluation infra missing from existing agent leaderboards.
- **Link:** https://arxiv.org/abs/2606.19338

## 2. Current World Models Lack a Persistent State Core
- **Authors:** Jinpeng Lu, Dexu Zhu, Haoyuan Shi, et al.
- **Summary:** Argues that today's video/world models render plausible frames but fail to maintain world state when objects leave the camera frame, and proposes WRBench, a benchmark that treats camera motion as an intervention on observability. Across 9,600 videos from 23 models the consistent failure is "tracking-shot" behavior — when a target reappears, models resume it in the state it was abandoned in rather than advancing the event that ran while unobserved.
- **Theme:** Memory for LLMs and agents (persistent state / world state)
- **Relevance:** Reframes a memory question — does state persist independently of observation? — as a measurable property of generative agents/world models; useful framing for any long-horizon agentic system.
- **Link:** https://arxiv.org/abs/2606.20545

## 3. S-Agent: Spatial Tool-Use Elicits Reasoning for Spatial Intelligence
- **Authors:** Yalun Dai, Hao Li, Shulin Tian, et al.
- **Summary:** Recasts spatial VLM reasoning as spatio-temporal evidence accumulation: a VLM planner decides what spatial evidence to gather, calls a hierarchy of 2D/3D grounding tools, and writes into a dual memory store — Scene Memory for evolving world state and Agent Memory for reasoning trace. Training-free use already improves open- and closed-source VLMs; SFT on the resulting trajectories (S-300K) yields S-Agent-8B which is reported to match GPT-5.4 / Gemini-3-class results on multi-view and video spatial benchmarks.
- **Theme:** Memory for agents + multi-agent/tool orchestration
- **Relevance:** Combines explicit episodic memory with tool-use orchestration — sits cleanly at the memory ∩ multi-agent intersection that this digest targets.
- **Link:** https://arxiv.org/abs/2606.20515

## 4. Playful Agentic Robot Learning (RATs — Robotics Agent Teams)
- **Authors:** Junyi Zhang, Jiaxin Ge, Hanjun Yoo, et al. (incl. Ken Goldberg, Trevor Darrell, Ion Stoica, Angjoo Kanazawa)
- **Summary:** Proposes a multi-agent coding harness where a team of LLM agents proposes its own play tasks, writes Code-as-Policy programs, verifies/retries with step-level feedback, and distills successful runs into a persistent code-skill library used at test time. Reports +20.6 and +17.0 pp gains on LIBERO-PRO and MolmoSpaces over no-play and random-play baselines, and shows the learned skills transfer into other CaP agents (+8.9 / +8.8 pp on RoboSuite and real-world) by retrieving them into context — no finetuning.
- **Theme:** Multi-agent frameworks & orchestration
- **Relevance:** Multi-agent role assignment plus a persistent, retrievable skill memory — a clean architecture for plug-and-play agent capabilities.
- **Link:** https://arxiv.org/abs/2606.19419

## 5. ENPIRE: Agentic Robot Policy Self-Improvement in the Real World
- **Authors:** Wenli Xiao, Jia Xie, Tonghe Zhang, et al. (incl. Ken Goldberg, Linxi "Jim" Fan, Yuke Zhu, Guanya Shi)
- **Summary:** A four-module harness — Environment (auto-reset/verify), Policy-Improvement, Rollout (single or fleet), and Evolution (an agent that reads logs, consults literature, and edits the training code) — that closes the real-world reset/execute/verify/refine loop for coding agents. With this harness, frontier coding agents autonomously train dexterous manipulation policies to 99% success on tasks like organizing a pin box, fastening a zip tie, and tool use, and the loop accelerates when agents are dispatched across a robot fleet.
- **Theme:** Multi-agent frameworks & orchestration
- **Relevance:** Concrete, modular agent orchestration architecture for autonomous research loops; directly relevant to how to build agentic harnesses with clear role separation.
- **Link:** https://arxiv.org/abs/2606.19980

## 6. FAPO: Fully Autonomous Prompt Optimization of Multi-Step LLM Pipelines
- **Authors:** Paul Kassianik, Baturay Saglam, Huaibo Zhao, et al.
- **Summary:** Hands Claude Code a standardized pipeline codebase and lets it iterate: evaluate, inspect intermediate steps, attribute failures, propose scoped prompt edits, and only escalate to structural chain changes when prompt-only attribution flags a structural bottleneck. Beats GEPA on 15/18 model×benchmark cells with a +14.1 pp average gain, and wins all six HoVer/IFBench cells where structural edits were triggered with +33.8 pp.
- **Theme:** Multi-agent frameworks & orchestration (LLM-pipeline optimization)
- **Relevance:** Shows a coding-agent-driven loop optimizing both prompts and chain structure of multi-step LLM systems — directly relevant to agent-stack engineering.
- **Link:** https://arxiv.org/abs/2606.19605

## 7. Native Active Perception as Reasoning for Omni-Modal Understanding (OmniAgent)
- **Authors:** Zhenghao Xing, Ruiyang Xu, Yuxuan Wang, et al.
- **Summary:** Formulates long-video understanding as a POMDP and trains OmniAgent to interleave Observation, Thought, and Action steps that selectively fetch audio-visual evidence rather than dumping the whole video into context. Uses agentic SFT plus an RL phase with a credit-assignment scheme (TAURA), and reports a 7B OmniAgent outperforming a 72B Qwen2.5-VL baseline.
- **Theme:** Multi-agent / agentic frameworks (active perception loop)
- **Relevance:** Decouples reasoning cost from input length via an explicit perception-as-action loop — a useful design pattern for long-context agents.
- **Link:** https://arxiv.org/abs/2606.19341

## 8. STARE: Surprisal-Guided Token-Level Advantage Reweighting for Policy Entropy Stability
- **Authors:** Haipeng Luo, Qingfeng Sun, Songli Wu, et al.
- **Summary:** Targets the entropy-collapse failure mode of GRPO-style RL fine-tuning by reweighting token-level advantages using batch-internal surprisal quantiles, and adds a closed-loop target-entropy controller. Reports 4–8% gains over baselines on AIME24/25 across 1.5B–32B model scales, with cleaner entropy trajectories.
- **Theme:** Core LLM advances (RL post-training stability)
- **Relevance:** Practical fix for a known instability in modern RL-for-reasoning recipes; relevant to any agent stack that fine-tunes via GRPO.
- **Link:** https://arxiv.org/abs/2606.19236

## 9. Beyond Static Leaderboards: Predictive Validity for the Evaluation of LLM Agents
- **Authors:** Dhaval C. Patel, Kaoutar El Maghraoui, Shuxin Lin, et al.
- **Summary:** Aggregates fourteen parallel implementation studies on an MCP-based industrial agent benchmark and seven prior agent benchmarks, and argues aggregate-score leaderboards systematically mis-rank deployed agents because in-sample rank doesn't transfer out-of-distribution. Proposes ranking configurations by predictive validity — correlation between in-sample and held-out rank — and lays out a twelve-tier measurement apparatus plus three falsifiable OOD criteria.
- **Theme:** Core LLM advances (agentic-system evaluation methodology)
- **Relevance:** Methodological piece that complements the memory/multi-agent themes — tells you how to evaluate the systems the other papers propose.
- **Link:** https://arxiv.org/abs/2606.19704

## 10. Sumi: Open Uniform Diffusion Language Model from Scratch
- **Authors:** Mengyu Ye, Keito Kudo, Wataru Ikeda, et al.
- **Summary:** Reports what the authors describe as the first uniform-diffusion language model pretrained from scratch at the 7B / 1.5T-token scale, with full weights and recipe released. Competitive with autoregressive baselines on knowledge and reasoning benchmarks but lagging on commonsense — a useful open data point on whether non-AR LMs can be a backbone for future agent stacks.
- **Theme:** Core LLM advances (alternative LM architecture)
- **Relevance:** Open, reproducible non-autoregressive LM at meaningful scale — relevant background for anyone considering diffusion-style decoding in agent pipelines.
- **Link:** https://arxiv.org/abs/2606.19005
